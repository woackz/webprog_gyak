HTML & CSS alapok

Ismerd meg a webfejlesztés két alappillérét

HTML

A HTML (HyperText Markup Language) a weboldalak szerkezetének leírására szolgáló jelölőnyelv. Minden oldal HTML-ből indul ki, ez határozza meg a tartalmat, a szövegeket, a képeket és a szerkezeti elemeket.

CSS

A CSS (Cascading Style Sheets) a weboldalak megjelenéséért felelős. Ezzel szabályozhatjuk a színeket, betűtípusokat, elrendezéseket és az animációkat is. Lehet külön fájlban vagy beágyazva is használni.

A HTML és CSS kombinálása lehetővé teszi, hogy egyszerű, de esztétikus weboldalakat készítsünk. Érdemes először az alapokat megtanulni, majd ezekre építkezve haladni a bonyolultabb technikák felé. A Bootstrap keretrendszer például rengeteg előre elkészített elemet biztosít, ami megkönnyíti a munkánkat.
